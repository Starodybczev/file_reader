**File Reader** — универсальное веб-приложение для удобного просмотра документов и их автоматической сверки с техническим заданием. Больше не нужно переключаться между вкладками, чтобы проверить, всё ли вы учли.

[Посмотреть демо](#) | [Сообщить об ошибке](https://github.com/ваш-логин/file_reader/issues)

---

## 🚀 Основной функционал

* **Мультиформатность:** Поддержка файлов `PDF`, `Word (.docx)`, `HTML` и `JSON`.
* **Интеллектуальная сверка:** Загрузите файл и текст задания — приложение подсветит ключевые моменты и проверит соответствие.
* **Удобный UI:** Адаптивный интерфейс, написанный на чистом CSS/SCSS без тяжелых фреймворков.
* **Безопасность:** Обработка файлов происходит на стороне клиента, данные не отправляются на сервер.

---

## 🛠 Стек технологий

* **Core:** [TypeScript](https://www.typescriptlang.org/) — для надежной типизации и чистого кода.
* **Logic:** [JavaScript (ES6+)](https://developer.mozilla.org/en-US/docs/Web/JavaScript) — динамика и работа с файловыми API.
* **Styling:** [SCSS](https://sass-lang.com/) & [CSS3](https://developer.mozilla.org/en-US/docs/Web/CSS) — гибкая верстка и современные эффекты.
* **Markup:** [HTML5](https://developer.mozilla.org/en-US/docs/Web/HTML) — семантическая структура.
